# Editorial Policy

Truth World is a reader and translation layer. It is not an endorsement engine.

## Output rules

- Always preserve a visible source link when available.
- Keep the original text separate from AI output.
- Label AI-generated summaries as AI summaries.
- Extract claims without presenting them as verified facts.
- Default verification status to `Not independently verified` unless independent evidence is provided.
- Do not rewrite source material into campaign messaging, advocacy copy, or personal attacks.

## Language rules

- Summaries should be concise and neutral.
- Translations should preserve meaning without intensifying tone.
- If a phrase is ambiguous, preserve the ambiguity instead of resolving it editorially.

## Partnership claims

Do not claim operational support, endorsement, data access, or API partnership from Truth Social, xAI, SpaceX, Donald Trump, or Elon Musk unless written approval exists and the repository is updated with the approved integration path.
