# Legal Notes

This repository is not legal advice.

Truth World v1 is designed around compliant inputs:

- Demo fixtures.
- Manual user-provided text.
- Future approved partner/API feeds.

It intentionally avoids automated access to Truth Social. Truth Social's Terms of Service restrict automated use, robots, data mining, and similar data gathering or extraction tools. Any future live ingestion must be reviewed against the then-current terms and written partner/API permission.

The Grok/xAI API key and any Codex/OpenAI operator credentials must stay server-side. Do not place `XAI_API_KEY`, Codex auth material, or partner API credentials in browser JavaScript, static JSON, screenshots, commits, or public hosting configuration.

SpaceX operation is a business/partnership goal, not a current repository feature.
